# SL Programming Language

Простой компилируемый язык программирования с C-подобным синтаксисом. Реализован с использованием Flex, Bison и C++.

## Возможности языка

- **Типы данных**: `int`, `float`, `double`, `string`, `bool`, `void`
- **Функции**: С указанием типов возвращаемых значений через `->`
- **Классы**: Определение классов с конструкторами
- **Объектно-ориентированное программирование**: Инкапсуляция данных и поведения
- **Управляющие конструкции**: `if/else`, `while`, `for`, `do-while`, `switch/case`
- **Операторы**: Арифметические, логические, сравнения, присваивания, инкремент/декремент
- **Массивы**: Объявление и доступ к элементам
- **Константы**: Ключевое слово `const`
- **Комментарии**: Однострочные `//` и многострочные `/* */`
- **Библиотеки**: Компиляция в статические и динамические библиотеки

## Сборка

```bash
# Сборка всего проекта
make all

# Только компилятор
make slc

# Только менеджер проектов
make slpm

# Запуск тестов
make test

# Установка в систему
make install

# Удаление из системы
make uninstall
```

## Тестирование

Проект включает комплексный набор тестов:

- **basic_test.sl**: Базовые возможности языка
- **expressions_test.sl**: Арифметические выражения
- **control_flow_test.sl**: Управляющие конструкции (if, while, for)
- **functions_test.sl**: Функции, рекурсия, сложные вычисления
- **class_test.sl**: Классы и конструкторы
- **advanced_test.sl**: Продвинутые конструкции (циклы, управление потоком)
- **library_test.sl**: Создание библиотек

Все тесты автоматически запускаются командой `make test`.

## Использование

### Компилятор (slc)

```bash
# Компиляция в исполняемый файл
slc source.sl output

# Компиляция в динамическую библиотеку
slc source.sl -shared

# Компиляция в статическую библиотеку
slc source.sl -static
```

### Менеджер проектов (slpm)

```bash
# Создание исполняемого проекта
slpm new my_app

# Создание проекта библиотеки
slpm new my_lib --library

# Сборка проекта
slpm build

# Сборка и запуск
slpm run

# Просмотр проектов
slpm list

# Удаление проекта
slpm remove path
```

## Примеры кода

### Простая программа

```sl
function main() -> int {
    return 0;
}
```

### Функции с типами

```sl
function add(int a, int b) -> int {
    return a + b;
}

function main() -> int {
    int result = add(5, 3);
    return result;
}
```

### Управляющие конструкции

```sl
function main() -> int {
    int x = 10;

    if (x > 5) {
        return 1;
    } else {
        return 0;
    }
}
```

### Циклы и массивы

```sl
function main() -> int {
    int[] numbers = {1, 2, 3, 4, 5};

    for (int i = 0; i < 5; i++) {
        if (numbers[i] % 2 == 0) {
            return numbers[i];
        }
    }

    return 0;
}
```

### Классы

```sl
class SimpleClass {
    SimpleClass() {
        // Конструктор
    }
}

class ClassWithParams {
    ClassWithParams(int value) {
        // Конструктор с параметрами
    }
}

function main() -> int {
    SimpleClass obj1 = SimpleClass();
    ClassWithParams obj2 = ClassWithParams(42);
    return 0;
}
```

## Системные требования

- GCC или другой C++17 компилятор
- Flex (лексический анализатор)
- Bison (синтаксический анализатор)
- Linux/macOS/Windows

## Архитектура

SL компилируется в C код, который затем компилируется GCC в исполняемый файл или библиотеку. Это позволяет генерировать эффективный машинный код и обеспечивать совместимость с существующими C библиотеками.
